## 📝 Description
Ce projet est une application web conçue pour **centraliser et organiser mes recherches de stage**. L'objectif est de remplacer les tableurs Excel classiques par une interface dédiée, permettant un suivi plus fluide et structuré des candidatures.

**Pourquoi ce projet ?**
*   Quitter l'aspect rigide d'Excel.
*   Centraliser les contacts et les documents envoyés.
*   Avoir une vision claire de l'état d'avancement (Relances, Entretiens, Refus).

---

## ✨ Fonctionnalités
*   **Tableau de bord :** Vue d'ensemble de toutes les demandes effectuées.
*   **Gestion de statut :** Suivi en temps réel (En attente, Relancé, Entretien, Accepté, Refusé).
*   **Fiches détaillées :** Stockage des coordonnées des recruteurs et des liens des annonces.
*   **Interface Responsive :** Consultable sur PC comme sur mobile.

---

## 🛠️ Technologies utilisées  
*   **Frontend :** HTML5, CSS3 (Design personnalisé).
*   **Backend :** PHP 8.x.
*   **Base de données :** MaraiDB.
*   **Serveur :** Compatible avec Apache / WAMP / Laragon.
